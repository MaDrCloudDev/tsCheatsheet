Fixed lust where elements are the same data types (Vectors are growable Arrays).

## Use std::mem

Can use this instead (like an import) and then just use "mem" in the example measuring the memory size of the array

```
pub fn run() {
let mut numbers: [i32; 5] = [1, 2, 3, 4, 5];

    // Reassign a value (you can get numbers to mut, then reassign, however?)
    numbers[2] = 20;

    println!("{:?}", numbers); // errors if array isn't exactly 5 indices

    // Get single value
    println!("Single value: {}", numbers[0]);

    // Print array length
    println!("Array length: {}", numbers.len());

    // Arrays are stack allocated (get the memory of the array)
    println!("Array occupies {} bytes", std::mem::size_of_val(&numbers));

    // Get slice
    let slice: &[i32] = &numbers[0..2];
    println!("Slice: {:?}", slice)
}
```
