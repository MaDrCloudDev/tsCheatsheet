## Rust

```rust copy
fn greet(name: &str) -> String {
    format!("Hello, {}!", name)
}

fn main() {
    let person = "John";
    let greeting = greet(person);
    println!("{}", greeting);
}
```

## TypeScript

```ts copy
function greet(name: string): string {
	return `Hello, ${name}!`;
}

function main() {
	let person = 'John';
	let greeting = greet(person);
	console.log(greeting);
}
```
